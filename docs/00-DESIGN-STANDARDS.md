Home-Datacenter Design Standards

Purpose

This document defines the visual standards used throughout the Home-Datacenter project.

The goal is to ensure every diagram is consistent, easy to read, and professionally presented.

⸻

General Rules

* One diagram should answer one question.
* Keep diagrams simple and uncluttered.
* Use consistent spacing and alignment.
* Prefer clarity over decoration.

⸻

Shapes

Component	Shape
Internet	Cloud
Router	Rounded Rectangle
Switch	Rounded Rectangle
Servers	Rounded Rectangle
External Services	Cloud

⸻

Connectors

* Straight or orthogonal connectors only.
* Avoid crossing lines whenever possible.

⸻

Titles

Every diagram should include:

* Project Name
* Diagram Name

Example:

Home-Datacenter

Physical Infrastructure Overview

⸻

Naming

Draw.io source files:

physical-infrastructure.drawio

Exported diagrams:

physical-infrastructure.svg

⸻

Future Standards

These standards will evolve as the project grows.

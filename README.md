## 📖 About

Home-Datacenter is my personal infrastructure project where I design, build, secure, monitor, and document a self-hosted environment using industry-inspired engineering practices.

This project serves two purposes:

- Power my everyday self-hosted services
- Demonstrate my skills in Linux, networking, Docker, automation, and cloud engineering

Rather than simply running containers, I focus on understanding **why** technologies are used, documenting architectural decisions, and continuously improving the environment.

---

## 🎯 Goals

- Learn Linux administration
- Master Docker
- Build secure infrastructure
- Improve networking knowledge
- Practice documentation
- Learn automation
- Prepare for Cloud Engineering
- Build a portfolio that reflects real engineering practices
